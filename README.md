# RUBE

**RUBE — Ridiculously Universal Byzantine Exchange** is an experimental protocol for transmitting one authenticated Boolean by involving far more protocols, trust domains, serialization formats, and physical processes than the task can justify.

The normative specification is maintained as an Internet-Draft-style document:

- [RUBE/1.0 Protocol Specification](draft-rube-protocol-01.md)

RUBE is designed around one principle:

> Any operation that could be implemented as a function returning a Boolean can instead be implemented as a distributed, cryptographically witnessed, cross-protocol, print-mediated consensus ceremony.

## Status

RUBE/1.0 is an experimental specification. Implementation is permitted; deployment is inadvisable.
